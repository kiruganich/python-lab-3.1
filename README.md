#  Реализация очереди задач ( TaskQueue ) с ленивой обработкой — Лабораторная работа №3

## Описание

Реализация пользовательской коллекции `TaskQueue` с поддержкой протокола итерации, ленивых фильтров по статусу и приоритету, а также потоковой обработки задач.

## Основной функционал:

### Очередь задач ( TaskQueue )
- **Протокол итерации**: `__iter__` возвращает новый итератор при каждом вызове — поддерживает повторный обход
- **Ленивые фильтры**: `filter_by_status`, `filter_by_priority`, `filter_active` — генераторы, не создающие копии данных в памяти
- **Потоковая обработка**: `stream()` — отдача задач по одной через `yield from`
- **Управление очередью**: `add`, `extend`, `pop` (FIFO), `clear`
- **Специальные методы**: `__len__`, `__bool__`, `__contains__`, `__repr__`
- **Валидация**: проверка типов через `TaskValidationError` — единообразие с предыдущими лабораторными
- **Логирование**: все операции добавления, очистки и ошибок логируются
- **Совместимость**: работает с `for`, `list()`, `sum()`, `len()`, `in`, `sorted()`, `max()`, `min()`
- **Покрытие тестами**

## Установка и запуск

### 1. Требования
- Python 3.10+ (для синтаксиса `str | int`)
- pytest 7.0.0+

### 2. Установка зависимостей
```
pip install -r requirements.txt
```

### 3. Запуск демонстрации
```
python main.py
```

### 4. Запуск тестов
```
pytest tests.py
```
